# Arvsplan — publik demo

Statiskt demo-bygge av Arvsplan-frontenden där hela backenden simuleras i
webbläsaren (delade deterministiska motorer + localStorage). Ingen data
lämnar din webbläsare. AI-läge och PDF är avstängda i demon.

**Testa här: https://superheroin888.github.io/arvsplan-demo/**
Juristportal: lägg till `#jurist` (demo-konto eva.nord / jurist123).

Källkod: privata repona `arvsplan-frontend` + `arvsplan-backend`.
